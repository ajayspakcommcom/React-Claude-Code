# React App

A minimal React 18 + TypeScript + Webpack starter app.

## Tech Stack

- React 18
- TypeScript 5
- Webpack 5
- webpack-dev-server

## Getting Started

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm start
```

Opens at `http://localhost:8080` by default.

### Build for production

```bash
npm run build
```

Output is generated in the `dist/` directory.
